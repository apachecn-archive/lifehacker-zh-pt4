# 投影主屏幕

> 原文：<https://lifehacker.com/the-cast-shadows-home-screen-1682630707>

虽然南方公园可能已经走在了彩纸设计的前沿，但世界其他地方已经开始迎头赶上。今天的设计特点是现代，平面的外观，高对比度的颜色和大量的阴影。

*   [Zooper Widget](https://play.google.com/store/apps/details?id=org.zooper.zwpro)
*   [媒体实用程序](https://play.google.com/store/apps/details?id=com.batescorp.notificationmediacontrols.alpha)
*   [简单的 RSS 小工具](https://play.google.com/store/apps/details?id=de.j4velin.rssWidget)
*   [源文件](http://goo.gl/9D9Rhk)
