# Azul 主屏幕

> 原文：<https://lifehacker.com/the-azul-home-screen-1725442680>

今天的主屏幕设计使用了适度的蓝绿色模板和拼图一样的网格，给你的主屏幕一个充满活力和有条理的外观。



要在您的主屏幕上展示这种设计，您需要:

*   [Zooper Widget Pro](https://play.google.com/store/apps/details?id=org.zooper.zwpro)
*   [简单的 RSS 小工具](https://play.google.com/store/apps/details?id=de.j4velin.rssWidget)
*   [Zooper 文件](https://goo.gl/ckxdSx)

你有没有一个很棒的、被调整到遗忘的家或者你自己的锁屏，你想分享吗？把它贴在下面的评论里，或者贴在你自己的 Kinja 博客上，标签是“主屏幕展示”(没有引号)。一定要包括你是如何制作它的描述，这样我们就可以把它作为下一个特色主屏幕。
