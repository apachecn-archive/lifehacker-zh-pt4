# 火星双显示器桌面

> 原文：<https://lifehacker.com/the-martian-dual-monitor-desktop-1720857246>

利用 [MSL 好奇号](https://www.nasa.gov/mission_pages/msl/index.html) 拍摄的好看壁纸，一些对虚拟桌面的巧妙使用，以及一些桌面小工具，在 Windows 中组装了这个双显示器桌面。

Watch

自从上次我们看到你们喜欢的双显示器台式机，我们想我们应该强调一下另一个。Orbmiser 的桌面相当简单——是 Windows 7，运行一些停产的雅虎小工具，但不知何故，仍能在他的桌面上运行。以下是他使用的方法，如果你想在你的系统上得到同样的效果，你也需要这些方法:

*   [MSL 火星微明壁纸](http://i.imgur.com/1uV7XPz.jpg)
*   [Steam VS for Windows 7 主题](http://yorgash.deviantart.com/art/Steam-VS-for-Windows-7-449032822)
*   虚拟桌面的(您可以在右侧显示屏的左下方看到它们)

因为奥比索还在用雅虎！小工具，如果你愿意，你可以从仍然有安装程序的第三方来源获得它们(日历和天气小工具被捆绑在一起，所以技术上你仍然可以使用它们)，只是要小心，只从值得信赖的来源下载它们，因为该软件太旧了。如果你不想走那条路，我们的老朋友 [雨量计](http://rainmeter.net/) 可以给你一个类似的结果。如果你喜欢你所看到的，去看看 Orbmiser 的 Flickr 页面，链接如下。如果你是一个 Linux 用户，他甚至有一个类似的设置，使用 [KDE Plasema 5 运行在 Manjaro](https://www.flickr.com/photos/orb9220/19544766398/in/pool-lifehacker-desktop-showandtell/) 上！
