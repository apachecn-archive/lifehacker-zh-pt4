# 地面桌面

> 原文：<https://lifehacker.com/the-earthbound-desktop-1721828694>

读者 [bionicfemme](http://kinja.com/bionicfemme) 分享了这个桌面到我们的 [桌面展示区](http://kinja.com/tag/desktop-showcase) 我们喜欢它，部分是因为我们分享了对地球的爱。如果你像我们一样喜欢它，这里有一些你可以设置它的方法。

Watch

Bionicfemme 在这里使用 OS X，所以如果你想让你的 Mac 也有类似的外观，这里是你需要的:

*   [壁纸](http://wall.alphacoders.com/big.php?i=475456)
*   [über sicht](http://tracesof.net/uebersicht/)，一款我们之前介绍过的 Mac [定制 app](http://lifehacker.com/ubersicht-customizes-your-os-x-desktop-with-widgets-and-1591137635)
*   [简单时钟小部件](https://github.com/soberstadt/simple-clock-widget) 为 bersicht
*   [über sicht 天气小工具](https://github.com/felixhageloh/weather-widget)
*   [绑定字体包](http://earthboundcentral.com/2009/11/ultimate-earthbound-font-pack/) 自定义所有的显示字体
