# 线框均衡器桌面

> 原文:[https://life hacker . com/the-wire frame-equalizer-desktop-1707357871](https://lifehacker.com/the-wireframe-equalizer-desktop-1707357871)

这个漂亮的桌面来自于 Flickr 用户 [ImminentFate](https://www.flickr.com/photos/116237792@N02/) ，他显然想要一些动态和有趣的东西，但设置起来仍然很简单。他是这样设计的。

Watch

这个 Windows 桌面是一个很棒的壁纸，一些雨量计皮肤的组合，只是一些坚实的注意，以确保所有的元素都符合主题。如果你想要同样的外观，这里是你需要的:

*   [3D 多边形壁纸](http://www18.imagesandwallpapers.com/img/3d_polygon_wallpaper_18087.jpg) (本例中水平翻转)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   桌面上的 [为雨量计的明文皮肤](http://redsaph.deviantart.com/art/Cleartext-for-Rainmeter-old-version-392908186) 为正在播放的信息
*   雨量计的 [VisBubble 皮肤](http://madhoe.deviantart.com/art/VisBubble-for-Rainmeter-488601501) 用于屏幕左侧的圆形可视化器(静止时看不出来，但会随着音乐移动)
*   雨量计的 [谜主题](http://kaelri.deviantart.com/art/Enigma-103823591) 用于天气信息、电源和音量控制
*   自定义任务栏图标的 [Syzygy 图标包](http://linkz57.deviantart.com/art/Syzygy-A-work-in-progress-129297185) 和 [Ecqlipse 图标包](http://crystalxinfinity.deviantart.com/#/art/ecqlipse-2-PNG-59941546?hf=1)

ImminentFate 解释说，这个时钟是为这个桌面设计的定制时钟，但许多其他时钟皮肤，包括上面主题中包含的一些皮肤(如 Enigma ),也很适合。如果你有任何问题，或者想确保你的看起来和这个一样，请务必访问下面链接的 Flickr 页面，提出你的问题。让 ImminentFate 知道是我们派您来的，并且知道您喜欢台式机！
