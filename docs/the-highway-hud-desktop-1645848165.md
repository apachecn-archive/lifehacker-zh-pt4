# 高速公路平视显示器桌面

> 原文：<https://lifehacker.com/the-highway-hud-desktop-1645848165>

读者 [阿米塔布](http://amiba17.kinja.com/) 喜欢他的高速公路桌面，即使他最近与我们分享了 [救赎之路桌面](http://lifehacker.com/the-road-to-redemption-desktop-1626707594) ，我们也喜欢这个 Linux 桌面 stats 的位置看起来像一个真正的 HUD，一个在工作时容易被忽略的桌面。

Watch

这里是阿米塔布用来设置这一切。请注意，他使用的是 Ubuntu，但是您可以在任何 Linux 发行版上这样做:

*   [壁纸](http://www.chaoswallpapers.com/view/open_highway-1280x800.html)
*   [Conky](http://conky.sourceforge.net/) ，一款针对 Linux 系统的轻量级系统监视器
*   Conky 的 [HUD 主题在屏幕中央创建显示](http://www.lukor.net/2012/12/05/install-hgm-hudnet-conky-in-ubuntulinux-mintfedoraother-distro/)
*   [Numix](http://numixproject.org/) GTK 主题来调整界面的外观和感觉
*   将 [Numix 圆](http://numixproject.org/) 图标设置为替换系统图标
*   屏幕下方的 [面板](https://launchpad.net/plank) 应用启动器

这就是你所需要的。不需要太多的努力就可以将普通的 Linux 桌面变成更加个人化的工作方式。如果你对它的设置有任何疑问，或者遇到了麻烦，可以去他的 Flickr 页面(链接如下)询问。
