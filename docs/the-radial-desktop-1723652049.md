# 放射状桌面

> 原文：<https://lifehacker.com/the-radial-desktop-1723652049>

读者 [jonredcorn2](https://www.flickr.com/photos/134554762@N08/) 在 Flickr 上分享了这个超级干净的桌面，挤满了有用的信息，有快捷方式和提要，信息丰富但不太忙。他是这样设置的。



本周的桌面是 Windows——确切地说是 Windows 10，所以你可以放心，如果你已经跳跃和升级了，他使用的一切都将工作。以下是获得相同外观所需的条件:

*   [圆形七彩深色壁纸](http://hdwyn.com/circle_colorful_dark_background_hd-wallpaper-61148/)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [简单 53 皮为雨量计](http://hysteria27.deviantart.com/art/Simple53-447658404) 在屏幕左侧创建快捷启动器
*   雨量计的 [英格玛皮肤](http://kaelri.deviantart.com/art/Enigma-103823591) ，可能是右边的馈送和系统监视器
*   [鸭鸭搜索皮](http://flyinghyrax.deviantart.com/art/Duck-Duck-Search-217592502) 为雨量计中心为搜索栏
*   [径向时钟皮](http://pvoijen.deviantart.com/art/RadialClock-1-02-539127755) 为雨量计屏幕中央的时钟
*   左下方为雨量计的 [弧度线性可视化器皮肤](http://freezingclouds.deviantart.com/art/Radian-Linear-Visualizer-1-9-2-535341307) 为正在播放的指示器和可视化器效果
*   一个定制版本的 [灵魂皮肤](http://bradd9.deviantart.com/art/Soul-323678927) 为雨量计创建顶部的快速链接(我们认为！)

这个有很多部分，但都非常简单，最终结果看起来非常干净。你可能会认为彩色中心来自其中一个皮肤，但它实际上是壁纸，其他一切都围绕着它安排。如果你喜欢你所看到的，去下面链接的 flickr 页面，让 jonredcorn2 知道你喜欢他做的东西——或者如果你有关于如何自己制作的问题，在那里问他们！
