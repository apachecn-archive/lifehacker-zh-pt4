# 游戏室工作区

> 原文：<https://lifehacker.com/the-playroom-workspace-1579618197>

在家里带着孩子工作并不容易 ，但是拥有一个既能工作又能照看孩子的空间会有所帮助。今天的特色工作区从办公室无缝过渡到儿童区，到处都有丰富的创造力。



游戏室/办公室属于新国内博客的 Nerissa Goco 和 Peter Hoang。多亏了白色的家具和墙壁，玩具、书籍和电子产品的大杂烩都融入了这个明亮的空间。有一个挂艺术品的手机、一个家庭照片板和许多储物盒。(篮子和善意是许多家庭杂乱问题的救星。)

当孩子长大后，这个空间可以轻松转变为专用办公室。查看尼瑞莎和彼得的帖子，获取更多共享空间的照片。

如果你有自己的工作空间可以炫耀，请将它添加到我们的 [Lifehacker 工作空间展示和讲述](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) Flickr 池中，与我们分享。确保您包含的任何照片至少为 640x360。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区。
