# 自定 Mac Pro 工作区

> 原文：<https://lifehacker.com/the-custom-macbook-pro-workspace-1606118848>

今天的特色工作空间是一个极简主义的家庭办公室，有一个古怪的特点:在胡桃木桌子上有一个雕刻点，用于放置圆柱形的 Mac Pro 电脑。



谷歌翻译无法翻译日本建筑师 [Takara Maru](http://perspective08.blog14.fc2.com/blog-entry-104.html) 在他的定制桌上的帖子，但图片可能就是你所需要的。这是一个光滑的工作空间——一个站立的桌子——作为干净设计的一部分，它半隐藏了 Mac。(我不知道他会怎么做，如果他不得不更换电脑，苹果改变尺寸，或者他得到一台非圆柱形的电脑，但现在让我们欣赏一下工作空间。)在下面的链接里看到更多的图片和细节。

如果你有自己的工作空间可以炫耀，请将它添加到我们的 [Lifehacker 工作空间展示和讲述](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) Flickr 池中，与我们分享。确保您包含的任何照片至少为 640x360。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区。
