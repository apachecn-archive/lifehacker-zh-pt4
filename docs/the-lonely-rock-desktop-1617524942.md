# 孤独的摇滚桌面

> 原文：<https://lifehacker.com/the-lonely-rock-desktop-1617524942>

读者[few cent](http://revolutionaryl524.kinja.com/)向我们的 [桌面展柜](http://kinja.com/desktop-showcase) 提交了这张精美的桌面。没事的时候，它看起来很舒服，而且有足够的工作空间，它的配件都放在旁边。这是你设置它需要的东西。

Watch

要像 FewCents 一样将您的桌面提交到我们的桌面展示，只需将它发布到您的 Kinja 博客上，标记为“桌面展示”，我们就会看到它。现在，这是你为自己重新创造这一个所需要的:

*   [壁纸](http://wallbase.cc/wallpaper/2940947) 离 Wallbase
*   来自 DeviantArt 的 [bAero Windows 主题](http://www.deviantart.com/art/bAero-261536009)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [简单新星](http://www.deviantart.com/art/Simply-Nova-Ver-2-0-270551439) 皮肤为雨量计为天气、时钟、统计
*   左边 app 页签的 [SDT RUS](http://www.deviantart.com/art/SDT-RUS-172465563) 雨量计皮肤
*   用于 Windows 的 [RocketDock](http://rocketdock.com/) 应用程序启动器
*   [Kobhens Dock 主题](http://www.deviantart.com/art/The-Kobhens-Dock-121368707) 为 RocketDocl
*   [吉尔桑斯文本图标](http://www.deviantart.com/art/Gill-Sans-Text-Dock-Icons-154466636) 图标包取代了洛克多克中的图标
*   [图标包](http://magnumhearted.deviantart.com/art/Mac-Dock-Icons-The-iCon-151583728) 为垃圾桶图标

三个工具，以及对每个工具的一点点定制，这就是将这个看起来很漂亮的桌面组装在一起所需要的全部。左侧的应用程序选项卡会在必要时弹出，其中包含一些常用的应用程序，底部的 dock 可以配置为随时显示或隐藏。如果你在让你的设置看起来更好方面有困难，或者想知道更多关于 FewCents 如何修改皮肤以使其工作的信息，请直接访问下面的 Kinja 链接，提出你的问题或者赞美你。
