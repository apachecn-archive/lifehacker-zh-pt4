# 蓝色的冬季桌面

> 原文：<https://lifehacker.com/the-blue-winter-desktop-1659849077>

Flickr 用户如果你和我们一样喜欢这个外观，这里有一些你需要在电脑上安装的工具。

Watch

Yusuf 的桌面看起来很棒，虽然它可能很忙，但你可以调整这些工具，以适应你想要的 Windows 桌面的大小和规模。以下是设置此功能所需的内容:

*   [壁纸](http://www.hdwallpapers.in/cold_leaves-wallpapers.html)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [风华 2](http://lilshizzy.deviantart.com/art/Rainmeter-Elegance-2-244373054) 皮肤为雨量计为回收站和时钟
*   [初级](http://flyinghyrax.deviantart.com/art/Elementary-2-0-426588102) 皮肤为日期和天气
*   [银河套装](http://kop4.deviantart.com/art/Galaxy-Suite-Rainmeter-281706374) 皮肤为雨量计为桌面音乐播放器
*   桌面上 haxagonal 应用程序启动器和系统监视器的 [有机 2.0](http://crosseout.deviantart.com/art/organiC-2-0-for-Rainmeter-317885895) 皮肤

这就是你所需要的。它是一些组件，但不太重，你可以调整它们中的每一个来添加和包含你想要的桌面部件和组件。如果你在使用它时遇到了麻烦，或者你不确定如何设置一切，请访问 Yusuf 的 Flickr 页面(链接如下),提出你的问题并赞美你。让他知道是我们派你去的！
