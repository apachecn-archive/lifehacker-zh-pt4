# 蒸汽石板桌面

> 原文：<https://lifehacker.com/the-steam-slate-desktop-1640341482>

在 [我们的脸书页面](http://facebook.com/lifehacker) 上，熊猫阮(Panda Nguyen)在我们关于上周特色桌面的帖子的评论中分享了他的桌面——嗯，我们太喜欢它了，我们也必须特色它。它可能看起来像是内置的，但它不是——它只是非常适合。他是这样设计的。



就像我们提到的，它可能看起来像是 Steam 嵌入在这个 Windows 7 中(我们之前说过这是一个 Windows 8 桌面-它不是！)桌面，但它真的不是——它只是太适合了，乍一看，你可能会认为它是它的一部分。事实并非如此——它实际上只是在顶部打开的，这张桌面[的交替视图(带有《星球大战》粉丝会喜欢的新壁纸)证明了这一点。尽管如此，整个事情看起来很统一。以下是设置这一切所需的内容:](https://www.flickr.com/photos/duong9a9ntt/15341475002/in/photostream/lightbox/)

*   [玩家国度壁纸](http://rog.asus.com/wp-content/uploads/2012/02/ROG_Black_Metal_InnervateD.jpg) (本集第 43 期)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [环皮](http://kaelri.deviantart.com/art/Rings-138547375) 为雨量计顶部对面的显示
*   左边是雨量计的 [英格玛皮肤](http://www.deviantart.com/art/Enigma-103823591) 用于新闻订阅和日历，右上方是系统监视器和实用程序
*   屏幕下方的应用启动器
*   [任务栏消除器](http://www.aviassin.com/taskbareliminator) 隐藏 Windows 任务栏(并且只能通过键盘命令恢复)
*   Steam 的[Metro](http://metroforsteam.com/)皮肤帮助应用程序更好地适应 Windows 8 的外观和感觉

这就是全部了。这本书没有太多关于皮肤和工具的内容，它既有知识性又不会让人分心。如果你很难让你的设置看起来如此，或者想知道更多关于如何修改一切以这样工作的信息，只需前往下面的 Flickr 链接，提出你的问题或赞美你。
