# 导航器的桌面

> 原文：<https://lifehacker.com/the-navigators-desktop-1702429376>

读者 [盐和胡椒](http://kinja.com/salt-and-pepper) 向我们的桌面展示区提交了这个漂亮、实用的桌面。边上的一些小部件和左边的工作空间，一切都完成了。这是它的制作方法。

Watch

这是 salt_and_pepper 的第一个定制桌面，我们可以说这是一个巨大的成功。壁纸是不错的选择，但是这里选择的雨量计也是不错的选择。如果你想把这个变成你自己的，这是你需要的。

*   [壁纸](http://lbc9.com/digital-art/fractals/black-and-white-fractals/wallpapers-3.html)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   雨量计的 [英格玛皮肤](http://enigma.kaelri.com/downloads/) 用于天气、音符部分、正在播放、音量和 CPU 监视器
*   [风华二皮](http://lilshizzy.deviantart.com/art/Rainmeter-Elegance-2-244373054)为雨量计为时间和日期
*   雨量计的 [Omnimo 皮肤](http://omnimo.info/)收件箱通知小部件的
*   [Windows 7 开始按钮改变器](http://www.thewindowsclub.com/windows-7-start-button-changer-released) 自定义开始按钮
*   [垃圾箱](http://www.1upindustries.com/bins/) 对 Windows 任务栏中的图标进行分组
*   [Ecqlipse 2 图标包](http://chrfb.deviantart.com/art/ecqlipse-2-PNG-59941546) 自定义任务栏图标

你不必像 salt_and_pepper 在这里做的那样调整任务栏，尽管我们可以清楚地看到任务栏中的黑色图标在这种壁纸和设置下比标准的多色图标看起来更好。如果你有任何问题或者想知道更多关于这是如何建立的，请到 salt_and_pepper 的 Kinja 帖子(链接如下)询问！
