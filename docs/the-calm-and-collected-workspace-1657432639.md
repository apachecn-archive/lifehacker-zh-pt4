# 平静而从容的工作空间

> 原文：<https://lifehacker.com/the-calm-and-collected-workspace-1657432639>

这是一个有个性的传统工作空间。从精美的木质桌面到海星拉手，正是这些小细节让这间办公室看起来浑然一体。



Viscusi Elson 室内设计公司设计了这间办公室，选择了以白色和木质为主的配色方案。蓝绿色的墙漆(本杰明·摩尔 HC-138 考文顿蓝色蛋壳，供参考)增加了一种平静的感觉，而深色的桌面在通风的房间中增加了巨大的反差。我喜欢这间办公室看起来像是有人实际使用的，同时又有一种时尚感。

如果你有自己的工作空间可以炫耀，请将它添加到我们的 [Lifehacker 工作空间 Show and Tell Flickr pool](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) 与我们分享。确保您包含的任何照片至少为 640x360。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区。
