# 侧边栏小部件主屏幕

> 原文：<https://lifehacker.com/the-sidebar-widgets-home-screen-1585636440>

Android:如果你有一部大手机和一双小手，那么在手机上到处伸手可能会很麻烦。我的彩屏用户 JRS05 的这个主屏幕把你需要的一切都放在一边，方便访问(抱歉左撇子)。



这种设计有五种不同的颜色，看起来都很棒。你甚至可以通过点击侧边栏中间相应的按钮来展开小的天气和音乐控件。

这个主屏幕是 Themer 应用程序的定制设计。您可以下载。zip 文件包含来自下面的源链接的主题。要安装它，请遵循以下说明:

1.  移动。zip to SD card 0 > my color screen > Themer > Exported > zip(您的初始位置可能会有所不同。)
2.  打开主题并浏览主题。
3.  在“我的主题”下，选择“导出”
4.  选择“jrs005.zip”
