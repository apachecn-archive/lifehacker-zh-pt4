# 文艺复兴桌面

> 原文：<https://lifehacker.com/the-renaissance-desktop-1582917082>

Flickr 用户 [AFGDesign](https://www.flickr.com/photos/86468800@N02/) 有一个很棒的工作空间*和*一个很棒的桌面。以下是他如何设置这一切，以及你需要在家里复制它。



这不是 AFGDesign 唯一的桌面，我们可能会在未来突出他的一些其他桌面，但我们喜欢这个，尤其是壁纸。很忙，但也很有创意。看着它，我们就想开始工作。无论如何，下面是让你的 Windows 机器看起来像这样所需要的:

*   杰米·格力的 j3concepts 中的 [壁纸](http://fcoar.neiio.ca/) (向下滚动，壁纸收藏在底部是一个 ZIP 文件)
*   Windows 8.1 的 [CRY-ML 视觉主题](http://cu88.deviantart.com/art/CRY-ML-Theme-For-Windows-8-1-446163055)
*   [共生图标包](http://winempyrean.net/symbiosis/)
*   用下载的图标替换系统图标的 [7 主题源补丁](http://www.xptsp.com/fixit/7tsp.php) (又名 7tsp)
*   [RocketDock](http://rocketdock.com/download) 在屏幕下方创建一个停靠发射器
*   [dock 中应用程序的简单圆形图标](http://www.iconspedia.com/pack/simple-round-icons-4375/)

这是我们在一段时间内看到的第一个不需要 Rainmeter 或其他类似的主题化或监控引擎的 Windows 桌面，但它看起来仍然很棒。如果你想知道他是如何设置的，或者你不能让组件正常工作，请访问他的 Flickr 页面，链接如下，提出你的问题。让他知道是我们派你来的，还有你有多爱他的作品！
