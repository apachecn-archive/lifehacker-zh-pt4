# 明亮的主屏幕

> 原文：<https://lifehacker.com/the-brights-home-screen-1621884331>

如果你合理利用手机空间，你可以只用一个主屏幕。这就是我的彩色屏幕用户 [双色调](http://mycolorscreen.com/duotoned/) 想要的，这种外观将你需要的一切都放在一个屏幕上

Watch

要在您的手机上获得这种外观，您需要:

*   [新星发射器](https://play.google.com/store/apps/details?id=com.teslacoilsw.launcher)T3】
*   [Zooper Widget Pro](https://play.google.com/store/apps/details?id=org.zooper.zwpro)
*   [Zooper Widget 文件](https://www.dropbox.com/sh/vidtzykh3rymos2/AADZPis6fqwCETmFIzErGckHa?lst)
*   [Popup Widget](https://play.google.com/store/apps/details?id=com.ss.popupWidget)
