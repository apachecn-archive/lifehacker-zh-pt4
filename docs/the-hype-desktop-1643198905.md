# 炒作桌面

> 原文:[https://lifehacker.com/the-hype-desktop-1643198905](https://lifehacker.com/the-hype-desktop-1643198905)

读者 [耶兹拉](http://yezra.kinja.com/) 向我们的 [桌面展示](http://kinja.com/desktop-showcase) 提交了这款桌面，指出他是简单、干净桌面的忠实粉丝，并最终建造了一款他们认为合适的桌面。你的不需要看起来完全一样，但是如果你喜欢要点，这里是你需要自己做的。

Watch

要像 Yezra 那样将您的桌面提交到我们的桌面展示，只需将它发布到您的 Kinja 博客上，标签为“桌面展示”，我们就会看到它！如果你让我们知道你是怎么做的，我们可以做专题报道！如果你想试着做一个这样的，这是你需要的:

*   来自 DeviantArt 的 [壁纸](http://adamt4050.deviantart.com/art/Hype-typography-wallpaper-331068197)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [进化音乐播放器](http://whimsy3sh.deviantart.com/art/Evolution-Music-Player-1-2-371577439) 为雨量计右上角圆形播放器
*   [初等皮肤](http://flyinghyrax.deviantart.com/art/Elementary-2-0-426588102) 右下角为雨量计的时间和日期
*   屏幕左侧全文本启动器雨量计的 [启动器 2.1.5](http://rainbowhoovez.deviantart.com/art/Launcher-2-1-5-408325917)
*   [画出字体](http://www.dafont.com/painted.font) 将整件事绑在一起

这就够了。有了字体，你可以改变颜色或用你想让它说的任何东西制作你自己的壁纸，或者你可以走这个方向，每次坐下来做一点工作就兴奋起来。此外，整个设置足够干净，当你打开窗户开始工作时，不会分散注意力或难以操作。如果你有关于设置的问题，或者需要帮助来使一切变得更好，去 Yezra 的 Kinja 博客(链接如下)问你的问题或者赞美你！

你有一台自己的好看又实用的台式机可以炫耀吗？分享给我们吧！使用标签 [**将它发布到你的个人 Kinja 博客**](http://kinja.com/tag/desktop-showcase) 或添加到我们的 [**Lifehacker 桌面展示并告诉 Flickr pool**](http://www.flickr.com/groups/lifehacker-desktop-showandtell/) 。截图必须至少为 640x360，并请包括您使用的信息，链接到您的壁纸，皮肤和主题，以及任何其他相关的细节。如果你的超棒的桌面吸引了我们的眼球，你可能会成为我们的特色！
