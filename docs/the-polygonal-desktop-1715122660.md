# 多边形桌面

> 原文：<https://lifehacker.com/the-polygonal-desktop-1715122660>

读者 [普什卡](http://kinja.com/pushkardesai999) 向我们的 [桌面展柜](http://kinja.com/tag/desktop-showcase) 提交了这个好看的桌面，我们喜欢它的样子。作为第二个屏幕或工作时间时有用，以下是他如何设置它。

Watch

普什卡的 Windows 桌面很容易安装。您将需要我们之前提到的基础知识，但以下是所有组件:

*   来自 DeviantArt 的 [多边形壁纸](http://www.deviantart.com/art/Polygon-01-514131710)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   屏幕左下方的 [极简皮肤](http://dashie36.deviantart.com/art/Minimalistic-System-Info-v2-333556546) 为雨量计创建系统监视器
*   页面右下角的 [Monstercat 可视化工具](http://www.deviantart.com/art/Monstercat-Visualizer-for-Rainmeter-1-2-1-486330771) 为音乐同步可视化工具
*   雨量计的 [子母钟](http://mirix10.deviantart.com/art/Zirim-Clock-472156048) 为屏幕中央的大时钟和日期
*   [简单媒体](http://lilshizzy.deviantart.com/art/Rainmeter-Simple-Media-1-1-212678796) 皮肤为屏幕左上角的温度
*   屏幕右上角的为模拟时钟
*   [RocketDock](http://rocketdock.com/) 作为替换的 app 启动器出现在屏幕下方
*   用 [乌瑞亚图标包](http://www.deviantart.com/art/Ourea-Icons-339972848) 来替换 RocketDock 中的默认图标

大概就是这样。如果你喜欢我们在很多台式机上看到的“屏幕中央的时钟”这种东西，你可以按照上面的方法设置它。如果没有，可以省略 Zirim 时钟皮肤，让屏幕中央保持开放。我们知道为什么 Pushkar 保留了它——当你没有任何东西时，它很棒，可能是你不经常使用的第二个屏幕或桌面，但角落里仍然有一个模拟时钟，以防你需要在屏幕上打开应用程序，但仍然想知道时间(因为任务栏没有设置为显示。)
