# 几何桌面

> 原文:[https://lifehacker.com/the-geometric-desktop-1708310485](https://lifehacker.com/the-geometric-desktop-1708310485)

Flickr 用户当没有窗户或被窗户覆盖时，整个装置看起来很棒。他是这样设置的。

Watch

这是一台 Windows 机器，但考虑到任务栏是隐藏的，其他一切都是定制的，你可能真的看不出来。如果你喜欢这个造型，这里有你需要的:

*   [壁纸](http://www.meh.ro/wp-content/uploads/2013/02/meh.ro10965.jpg)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   左侧 [风雅主题为雨量计](http://lilshizzy.deviantart.com/art/Rainmeter-Elegance-1-0-214509019) 为时钟和日期显示
*   [蜂巢主题为雨量计](http://rainmeterhub.com/skins/honeycomb/) 创建右边的发射器
*   [这个脚本隐藏和取消隐藏任务栏](http://www.thewindowsclub.com/hide-taskbar-windows-7-hotkey) 用热键把任务栏挡在外面

这就是这个的全部内容。不是一吨移动部件，而是一个很好的结果，很容易复制。如果你喜欢这个，或者对它是如何制作的有疑问，请访问下面的 Flickr 链接，让 eriksseibutis 知道你喜欢它的最终效果。让他知道是我们派你去的。
