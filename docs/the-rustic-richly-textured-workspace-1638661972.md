# 质朴、纹理丰富的工作空间

> 原文：<https://lifehacker.com/the-rustic-richly-textured-workspace-1638661972>

这座位于纽约市西村的阁楼式公寓以华丽的裸露砖墙、亮点色彩和复古家具为特色。

Watch

它属于 NYU 教授艾莉森·奥尔德里奇(美国参议员纳尔逊·w·奥尔德里奇的曾孙女)。她买下了三个相邻的工作室，合并成一套公寓，开放式客厅、餐厅和工作区的两端各有两间卧室，如上图所示。它温暖而多彩，看起来是一个生活和工作的好地方。你可以在下面的纽约时报幻灯片中看到更多的照片和细节。

如果你有自己的工作空间可以炫耀，请将它添加到我们的 [Lifehacker 工作空间展示和讲述](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) Flickr 池中，与我们分享。确保您包含的任何照片至少为 640x360。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区。
