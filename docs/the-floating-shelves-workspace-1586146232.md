# 浮动货架工作空间

> 原文：<https://lifehacker.com/the-floating-shelves-workspace-1586146232>

可调搁板系统可以真正改变房间的外观，通过灵活的搁板位置，让房间看起来更加开放和现代。今天的特色工作区有一个 [蒙德里安](http://en.wikipedia.org/wiki/Piet_Mondrian) 式的货架排列，看起来像一个明亮多彩的地方来完成事情。

Watch

这个办公室是由杰里米·莱文设计的。一摞摞的书，浅色的木头，长长的横排书架，还有关键配件大胆的颜色，真的把房间绑在了一起。另外，这里有太多令人惊叹的自然光。

如果你有自己的工作空间可以炫耀，请将它添加到我们的 [Lifehacker 工作空间展示和讲述](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) Flickr 池中，与我们分享。确保您包含的任何照片至少为 640x360。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区。
