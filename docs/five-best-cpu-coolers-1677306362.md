# 五个最好的 CPU 冷却器

如果您正在寻求超频、构建静音 PC，或者只是让您的设备以最高性能运行，您可能需要的不仅仅是为您的处理器提供库存冷却。有吨的售后 CPU 冷却器，但它可以很难选择一个好的。我们问过你，根据你的提名，这里有五个最好的选择。

### [扎尔曼 cnps 9500 a](http://www.zalman.com/global/product/Product_Read.php?Idx=277)T3】

Zalman 的 CNPS9500A 可能有点老，但你们许多人都称赞它高效、有效的冷却、安静的声音、好看的外观(如果你喜欢你箱子里的 LED 风扇)和紧凑的尺寸。它装有一个大而安静的 92 毫米风扇，三个热管(尽管 Zalman 说它的设计使它具有六个热管的传热能力)，以及一个超级安静的“静音模式”，风扇几乎完全静音，不会振动。散热器是纯铜的，整个装置的设计尽可能节省空间，因此您可以安装它而不会妨碍您的组件。它也很实惠——它会让你花费大约 [、亚马逊](https://www.amazon.com/dp/B000C0V5RU?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/five-best-cpu-coolers-1677306362&asc_source=&linkCode=ogi&psc=1&smid=ADYUHYFQRZHO0&tag=kinjalifehackerlink-20&th=1) 的 58 美元。

那些提名 H-110 的人特别提到了它的可靠性和易于安装，以及你永远不必担心泄漏的事实。你们中的一些人提到，这种设置非常适合保持超频处理器像运行股票速度一样酷，而另一些人指出，你已经在不同的平台上有了几个模型，包括 H-110，它们总是为你工作得很好。此外，这是一个坚实的，负担得起的水冷设置，是初学者友好的，很容易进入。更多 [可以在](http://lifehacker.com/vote-corsair-h-110-waterblock-why-i-highly-recommend-1676513961) 的提名帖中阅读。
