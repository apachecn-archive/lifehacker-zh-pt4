# 专业标签主页屏幕

> 原文:[https://lifehacker.com/the-pro-tabs-home-screen-1569577239](https://lifehacker.com/the-pro-tabs-home-screen-1569577239)

Android:Pro Tabs 为 Themer 设计了色彩鲜艳的标签和各种引人注目的照片，不仅让你的主屏幕看起来很好，而且让它保持整洁有序。

Watch

专业标签是一个自定义设计的主题。虽然它不是内置于应用程序本身，但你可以从下面的源链接从我的彩色屏幕手动下载它，并将其添加到主题。方法如下:

1.  单击下面来源处的绿色“下载主题文件”按钮。
2.  保存。压缩文件到你的手机。
3.  移动。zip to sdcard 0 > my color screen > Themer > Exported > zip*(您的初始位置可能会有所不同。)*
4.  打开主题并浏览主题。
5.  在“我的主题”下，选择“导出”
6.  选取“专业标签”
