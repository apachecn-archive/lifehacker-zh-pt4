# 消费者一直听到的关于维修的愚蠢论点

> 原文:[https://jalopnik . com/the-dumb-argument-consumers-heard-all-the-time-about-rep-1716987553](https://jalopnik.com/the-dumb-argument-consumers-hear-all-the-time-about-rep-1716987553)

23 年来，我一直在就缺陷汽车提起诉讼——包括柠檬法和其他法律。有一件事一直没有改变:我一遍又一遍地听到来自制造商和经销商的愚蠢的争论。

Watch

最近，我代表一位购买了一艘船的消费者提起诉讼，这艘船不在密歇根州的柠檬法范围内。他刚买不久，传动轴就失灵了。这根本行不通。他把它带回给卖它的商人，商人摆弄了一下，然后还给了他。他把它丢在水里，它不走了。他把它带回给了经销商。经销商又摆弄了几下，说已经修好了。

客户拿起它，重复这个过程。不去。经销商告诉我的客户，“它需要一个新的传动轴套，但制造商不会授权我们更换它。”他们说制造商只会授权他的人继续修理有缺陷的那个。当被问及是否可以在他的展厅从船上拿走一个时，卖家说不可以。

所以，我们提起诉讼。这就是预期防御的来源。制造商出现了，说:“如果你掉了西装，我们现在就更换传动轴套。”

这有多白痴？1.我委托人的船还在保修期内。他有权免费维修，所以在我们提起诉讼之前，“报价”是他有权得到的。2.卖家和制造商本可以在我们提起诉讼前修好船来避免这场诉讼。

然而，我总是听到这个。好消息是，在柠檬法案件中，制造商被告知，如果他们不抓住机会进行最后一次维修，将提起诉讼。即便如此，我也听到了。他们无视最后的机会信，我们提起诉讼，他们提出修理汽车，以换取解雇。就像我爸爸说的，“晚了一天，少了一美元。”

但这在非柠檬法领域更为常见。船，摩托车，房车，沙滩车等等。被告认为他们可以通过向你提供他们一直欠你的东西来逃避法律责任。客户拒绝这一点没有错。

除此之外，如果他们以前不能解决问题，谁会认为他们现在能解决呢？而且，提起诉讼需要时间和金钱。为什么消费者必须为此买单？(像 [马格努森-莫斯保证法案](http://carbuying.jalopnik.com/here-are-the-laws-you-need-to-know-if-you-bought-a-defe-1681535019) 这样的法律允许胜诉的消费者诉讼人从保证人那里收回律师费和诉讼费。)

有一两次，如果被告也同意支付律师费和诉讼费用，我的客户会同意这样的解决方案。但是被告不喜欢这样。有律师对我说，“你只是为了钱才这么做的。”

哦，你免费工作？

事实上，我做这个是为了钱，因为我喜欢在网上写这样的故事。我只是不告诉他们。
