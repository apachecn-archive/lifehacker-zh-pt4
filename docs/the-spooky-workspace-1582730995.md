# 怪异的工作空间

> 原文:[https://lifehacker.com/the-spooky-workspace-1582730995](https://lifehacker.com/the-spooky-workspace-1582730995)

如果你是恐怖电影的粉丝，你可能会喜欢今天的特色工作空间。虽然总体上装饰优雅，但克里斯托弗·布瑞尔的家庭办公室还装饰着一个骷髅面具、Baphomet 面具、弗莱迪·克鲁格手套和其他怪异的物品。

Watch

这间办公室并没有你想象中的那么恐怖，它来自为鬼屋提供设计服务的流氓谷。不过，如果你去看看下面 Brielmaier 的 Flickr photostream，你会发现很多有趣的东西(除了两台华硕 IPS 显示器、宜家书桌和蜂窝窗帘)。

如果你有自己的工作空间可以炫耀，请将它添加到我们的 [Lifehacker 工作空间展示和讲述](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) Flickr 池中，与我们分享。确保您包含的任何照片至少为 640x360。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区。
