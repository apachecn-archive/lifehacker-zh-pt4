# 火热的日落桌面

> 原文:[https://lifehacker.com/the-fiery-sunset-desktop-1637856217](https://lifehacker.com/the-fiery-sunset-desktop-1637856217)

Flickr 用户 [xhuxhaxhu](https://www.flickr.com/photos/38377466@N03/) 以前也曾在这里展示过桌面，但这一款以其亮橙色和暗黑色、开放的空间以及几个有用的屏幕小工具让她保持信息畅通而让我们印象深刻。她是这样设置的。

Watch

xhuxhaxhu 的桌面引人注目，但并不可怕地分散注意力。最棒的是，它不需要太多的设置。这个已经有一段时间了，但是这里是她用来在她的 Windows 机器上设置它的:

*   [壁纸](http://i.imgur.com/7Snosa6.jpg)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [英格玛皮肤](http://www.deviantart.com/art/Enigma-103823591) 为雨量计创建发射器(隐藏在上面)
*   雨量计的[Alli new](http://sn37.deviantart.com/art/Alli-Anew-353023363)皮肤创建上面的时钟、天气和 RSS 显示

这就是这个的全部内容。它并不复杂，而且看起来很棒——当然，您可以根据位置和 RSS 提要定制天气，以突出显示您选择的内容。如果你有麻烦让你的设置看起来更好，或者想知道更多关于 xhuxhaxhu 如何修改皮肤的工作方式，只需到下面的 Flickr 链接来问你的问题或者赞美你。
