# 操作系统 LinuX 桌面

> 原文：<https://lifehacker.com/the-os-linux-desktop-1623539448>

读者 [奥利弗](http://litech.kinja.com/) 想让他的 Linux Mint 桌面看起来尽可能像 Mac，这样其他人会觉得它很容易使用。鉴于我们以前的一些 Linux 特色桌面，我们知道这并不困难，但最终结果看起来仍然很棒。这是所有设置的方式。



要像 Josh 那样将您的桌面提交到我们的桌面展示，只需将它发布到您的 Kinja 博客上，标记为“桌面展示”，我们就会看到它。现在，这是你为自己重新创造这一个所需要的:

*   [壁纸](https://docs.google.com/file/d/0B_9kBKEmPm0jUC1oRFlHZTY1TGc/edit)
*   [库普弗](http://engla.github.io/kupfer/) 应用启动器
*   屏幕下方的 [开罗停靠](http://glx-dock.org/)
*   [黑色风华肉桂主题](http://gnome-look.org/content/show.php?content=147807)

这就是全部了。Oliver 正在运行 Linux Mint，但是我们认为你可以在任何地方，任何你认为合适的 Linux 版本中使用这些调整。最终的结果是一个操作系统，没有可怕的检修或返工，但看起来仍然像任何人都可以坐下来开始工作。如果你对此有疑问，或者想知道更多关于他是如何设置的，去他的 Kinja 博客(链接如下)问你的问题。
