# 灭火桌面

> 原文:[https://life hacker . com/the-fire-suppression-desktop-1697817553](https://lifehacker.com/the-fire-suppression-desktop-1697817553)

我们的朋友 [joergermeister](https://www.flickr.com/photos/joergermeister/) 做了不少我们喜欢的台式机，这款也不例外。干净，稀疏，但仍然有效，这是他如何设置的。

Watch

Joergermeister 正在使用 Windows 10 技术预览版，但你可以在任何 Windows 桌面上使用它。以下是他使用的方法，以及您需要下载的内容，以便对您的机器进行同样的操作:

*   [红色](https://www.flickr.com/photos/simon_vg/16590819635/sizes/l) 由 [西蒙【范海尔】](https://www.flickr.com/photos/simon_vg/)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [为雨量计最小的一个天气皮肤](http://mistrjosh.deviantart.com/art/MinimalOne-Weather-1-0-510234680) 为日期和天气显示
*   [Google Now 皮肤用于雨量计](http://vinod221091.deviantart.com/art/Google-now-410080356) 用于时钟和音乐播放器
*   屏幕下方的 [框停靠应用启动器的雨量计](http://dilshad9692.deviantart.com/art/Boxes-dock-1-0-519702491)
*   任务栏图标的 [Simplynotblack](http://ibaaan.deviantart.com/art/Simplynotblack-459372954) 图标包

一旦你在墙纸周围设置好了一切，你的设置应该看起来就像这样。幸运的是没有太多的活动部件。如果你有困难，或者只是想告诉 joergermeister 你有多喜欢他的作品，请点击下面的链接！
