# 晴空桌面

> 原文:[https://lifehacker.com/the-clear-skies-desktop-1613651245](https://lifehacker.com/the-clear-skies-desktop-1613651245)

Ami Banerji 的 Linux 桌面干净、简单、相当开放。它在小部件中放弃的东西在简单性中得到了回报，这是一件好事。以下是如何让您的电脑拥有相同外观的方法。

Watch

Ami 运行的是[Ubuntu 14.04“Trusty Tahr”](http://www.ubuntu.com/)，所以虽然一些外观和感觉是 Ubuntu 特有的，但并不是所有的都是，你可以在任何 Linux 发行版中获得类似的外观。以下是您需要的组件:

*   [壁纸](http://www.wallpaperup.com/274723/blue_gradient_skies_cloud.html) 来自壁纸
*   [Conky](http://conky.sourceforge.net/) ，一款针对 Linux 系统的轻量级系统监视器
*   [哈曼](http://zagortenay333.deviantart.com/art/Conky-Harmattan-426662366) 为康奇
*   [QlockTwo](http://pastebin.com/wK7JmG9H) 作者不是别人，正是 [okubax](https://www.flickr.com/photos/okubax/) ，他的作品我们之前已经在这里介绍过了
*   [Numix](http://numixproject.org/) GTK 主题
*   [Plank](https://launchpad.net/plank) ，一个用于 Linux 的应用 dock，用于屏幕下方的启动器
*   将 [Numix 方形](http://numixproject.org/) 图标设置为替换系统图标

与许多类似的设置一样，美妙之处在于您可以将它们都拿来并复制外观和感觉，或者您可以挑选对您有吸引力的东西并将其应用到您自己的系统中。如果你更喜欢底部的 dock 和它的图标，你可以选择它。如果你喜欢 Conky 配置，只需把它留下来。你可以做你选择的任何一个。如果你有问题或者只是想告诉 Ami 你有多喜欢桌面，请到下面链接的 Flickr 页面来赞美你并提出你的问题。
