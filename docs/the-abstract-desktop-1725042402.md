# 抽象桌面

> 原文：<https://lifehacker.com/the-abstract-desktop-1725042402>

giacomolawrence 的桌面非常有趣——一些抽象的形状，屏幕上活跃的小部件，以及需要时容易看到的快捷方式和信息。他是这样设置的。

Watch

这个桌面安装起来并不难，你只需要合适的组件。这是窗户，这是你应该拿起的东西:

*   [MD 蓝色大理石壁纸](http://simpledesktops.com/browse/desktops/2014/feb/11/md-blue-marble/) 作者 [马丁·多施](http://www.martindoersch.at/)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [径向时钟](http://pvoijen.deviantart.com/art/RadialClock-1-02-539127755) 皮为雨量计左上方为时钟
*   屏幕左下方的 [为天气雨量计的简单媒体皮肤](http://lilshizzy.deviantart.com/art/Rainmeter-Simple-Media-1-1-212678796)
*   屏幕中央的 [VIsBubble 皮肤为雨量计](http://undefinist.deviantart.com/art/VisBubble-for-Rainmeter-488601501) 为环绕地球的可视化器
*   屏幕右侧 app launcher 的 [雨量计](http://apiium.deviantart.com/art/Honeycomb-467211707) 蜂窝皮肤
*   屏幕右上角的 [风华 2 肤](http://lilshizzy.deviantart.com/art/Rainmeter-Elegance-2-244373054) 为雨量计为日期

这一次到此为止。有几个移动的部分，但结合起来看起来很棒，可以完成工作，更不用说他们利用一个很好的壁纸为工作空间添加一些动态光晕。如果你对它是如何完成的，或者如何让你自己的设置看起来和工作起来都像它一样有问题，请查看下面链接的 GiacomoLawrance 的 Flickr 页面——在那里你会想问你的问题。让他知道是我们派你去的！
