# 天际线桌面

> 原文:[https://lifehacker.com/the-skyline-desktop-1573155728](https://lifehacker.com/the-skyline-desktop-1573155728)

我们之前已经强调过一些的桌面，但是这个拱形的 Linux 桌面看起来很有趣，不管有没有壁纸。另外，它很容易配置。这是你需要的。

Watch

我们喜欢 Arch——事实上，我们为任何有兴趣尝试的人提供了一个完整的 Arch 指南。至于这个桌面，现在你需要让你的桌面看起来像 okubax 的:

*   来自 DeviantArt 的 [壁纸](http://brebenel-silviu.deviantart.com/art/Windows-Blue-Milestone-1-Wallpaper-2-355936745)
*   [Conky](http://conky.sourceforge.net/) ，一款针对 Linux 系统的轻量级系统监视器，用于定制时钟
*   [Conky-Harmattan](http://zagortenay333.deviantart.com/art/Conky-Harmattan-426662366)Conky 主题在桌面上创建信息栏
*   来自 DeviantArt 的 [Hex 等离子和 QtCurve 主题](http://garthecho.deviantart.com/art/Hex-439924084)
*   [Flattr 图标主题包](http://deviantn7k1.deviantart.com/art/Flattr-376383397) 自定义桌面图标

真的就这些了。当然，您不一定要运行 Arch 才能得到这个设置，但是它确实很有帮助。如果你对他如何设置一切感到好奇，或者如果你对事情如何运作有疑问，请前往奥克斯的 Flickr 页面询问你的问题。让他知道你有多喜欢他的作品！
