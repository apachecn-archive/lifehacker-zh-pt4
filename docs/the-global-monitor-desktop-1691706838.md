# 全局监视器桌面

> 原文：<https://lifehacker.com/the-global-monitor-desktop-1691706838>

[杰罗姆](http://jeromemasson.kinja.com/) 对他的工作笔记本电脑没有管理员权限(这是有道理的)，但他不想让这阻止他给它添加一点个性。结果是这个好看的，信息丰富的，自定义的 Windows 桌面。他是这样设计的。

Watch

对杰罗姆来说幸运的是，Rainmeter 和 RocketDock 不需要管理员权限就可以运行。结合使用它们，他成功地用 dock 替换了任务栏，定制了壁纸，使其与同事相比不会显得太格格不入，并做出了一些真正特别的东西。请记住，他正在调整他公司发行的壁纸，所以如果你想要确切的外观，你必须找到一个合适的替代品——他的不可用。除此之外，你还需要:

*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   雨量计 的 [GoogleBar 皮肤，调整为使用 DuckDuckGo](http://nardoxic.deviantart.com/art/GoogleBar-201690255)
*   [为雨量计的奇巧时钟皮肤](http://ddcraft.deviantart.com/art/Rainmeter-KitKat-Clock-413862679) 为桌面上方中央的浮动时钟
*   桌面右侧的 [简表三层雨量计](http://white-baron.deviantart.com/art/Simplesentencethree-175994306) 用于天气显示和系统监视器
*   Windows 的 [RocketDock](http://rocketdock.com/download) 启动器和最小化任务栏
*   [小图标设置](http://www.iconarchive.com/show/junior-icons-by-treetog.html) 来替换 RocketDock 中的默认图标

这就是这个的全部内容。你可以在任何壁纸上应用它，杰罗姆稍微调整了颜色和字体，以匹配他公司桌面的配色方案，所以他不必改变它，但仍然得到了一些真正个性化的东西。如果你想知道他是如何做到这一切的，或者如果你遇到了麻烦，请查看他在下面的 Kinja 博客帖子了解更多细节。
