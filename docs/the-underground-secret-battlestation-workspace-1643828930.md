# 地下秘密战斗站工作区

> 原文：<https://lifehacker.com/the-underground-secret-battlestation-workspace-1643828930>

需要离开你的五个孩子去完成一些工作(或者读一些漫画书)？Imgur 用户 [vannatter](http://vannatter.imgur.com/) 知道这种感觉，所以他在他的地下室精心制作了这个超棒的办公室。

Watch

万纳特的工作空间被生物特征锁定，以防止孩子们进入，这样他就可以专注于完成任务，或者玩视频游戏。他在后面有一台电视和一个装满漫画书的柜子(未显示)，五台显示器(天哪)，桌子下面有一个小冰箱，边缘有一些严肃的环境照明。更不用说一些惊人的电缆管理。

你可以在下面的相册中仔细查看他的所有硬件，并在 [这个 Reddit 线程](http://www.reddit.com/r/battlestations/comments/19kfdp/my_tech_mancave/) 中了解更多关于工作区的信息。该专辑还列出了他使用的所有设备，包括显示器、显示器支架、鼠标、键盘、扬声器，甚至冰箱。
