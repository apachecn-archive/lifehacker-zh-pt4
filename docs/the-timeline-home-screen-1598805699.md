# 时间线主屏幕

> 原文：<https://lifehacker.com/the-timeline-home-screen-1598805699>

你的手机可以显示如此多的信息，让你做如此多的事情，你很难掌握所有的信息。这个主屏幕将您需要的一切都放在一个易于阅读的主屏幕上。

Watch

设计遵循合理的流程，满足您一天的需求。它从早上开始，显示当天的日程安排。在它的正下方是当天的天气，所以你会知道发生了什么以及如何穿衣。第三部分是你可能需要查看以了解当天活动的任何应用程序，最后是一个音乐播放器小部件。您需要的一切都在一个地方。

这个设计是为主题定制的。你可以下载必要的。zip 文件 [在这里](http://mycolorscreen.com/2014/06/22/time-line/) ，然后通过以下操作安装它:

1.  移动。zip to SD card 0 > my color screen > Themer > Exported > zip(您的初始位置可能会有所不同。)
2.  打开主题并浏览主题。
3.  在“我的主题”下，选择“导出”
4.  选择“Time-line.zip”
