# 整洁的“鸟巢”工作区

> 原文：<https://lifehacker.com/the-neat-crows-nest-workspace-1522164195>

今天的特色工作区是一个充满乐趣的空间。尽管很吸引眼球，但它组织得非常好，其他洁癖者可能会对它印象深刻。

Watch

Flickr 用户 Moonfa11 称之为“乌鸦窝”，可能是因为有一只乌鸦在监视这个场景。从悬挂的 iPad 和耳机，到收集所有线缆的黑匣子，以及装饰一新的定制 PC，你会想要前往 Moonfa11 的 photostream，拍摄更多鼓舞人心的照片。特别聪明的是，床在双显示器后面，可以旋转，这样他就可以在床上玩游戏或看电影了:
