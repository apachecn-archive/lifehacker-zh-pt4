# 简单的条纹桌面

> 原文:[https://lifehacker.com/the-simple-stripe-desktop-1615726469](https://lifehacker.com/the-simple-stripe-desktop-1615726469)

有时窗口小部件和工具栏就在那里，而其他时候简单和开放更有利于完成工作。读者 [乔希](http://joshuata.kinja.com/) 向我们的 [桌面展示](http://kinja.com/desktop-showcase) 提交了这个桌面，我们很喜欢它。简单，开放，但仍然是个人的。他是这样设置的。

Watch

要像 Josh 那样将您的桌面提交到我们的桌面展示，只需将它发布到您的 Kinja 博客上，标记为“桌面展示”，我们就会看到它。现在，这是你为自己重新创造这一个所需要的:

*   乔恩·沃金在 SimpleDesktops 制作的马提尼·罗西制服 壁纸
*   [GeekTool](http://projects.tynsoe.org/en/geektool/) 系统监视器和配置实用程序为 OS X 创建时钟，现在在桌面右侧播放小部件
*   [Spotify Now Playing geek let](http://www.macosxtips.co.uk/geeklets/music/spotify-now-playing/)创建 Now Playing 显示
*   [阿尔弗雷德](http://www.alfredapp.com/) ， [我们最喜欢的用于 Mac 的应用启动器](http://lifehacker.com/the-best-app-launcher-for-mac-5835412) 用于屏幕中央的应用启动器(只有当你按下 Cmd+Space 时才显示，否则不可见)
*   [约塞米蒂主题为阿尔弗雷德](http://blog.alfredapp.com/2014/06/12/an-os-x-yosemite-theme-for-alfred/) 打造全白、扁平的外观

我们知道 Alfred 是可定制和主题化的，我们也知道它是 [真正强大的](http://lifehacker.com/a-beginners-guide-to-mouseless-computing-with-alfred-1596198655) ，但在这里它被用作桌面的一个有吸引力的部分，它将所有东西聚集在一起，并保持所有东西整齐有序。我们想象 OS X 码头很可能被设置为自动隐藏，除非它的鼠标也在这里。如果你喜欢你所看到的，或者对 Josh 的设置有疑问，就去他的 Kinja 博客(链接如下)给他留言吧！让他知道是我们派你去的。
