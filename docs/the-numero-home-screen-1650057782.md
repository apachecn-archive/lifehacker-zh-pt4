# 数字主屏幕

> 原文：<https://lifehacker.com/the-numero-home-screen-1650057782>

今天的主屏幕对 AMOLED 特别友好。虽然最近 Android 的外观更加明亮，但我的彩屏用户 [21macro12](http://mycolorscreen.com/21MaRcO12/) 的设计坚持了黑暗的主题，只有一个主条纹是例外。



这种设计对所有发射器都是友好的。您可以从 [这里](http://www.android-tuning.sk/wp-content/uploads/Numero-themer.zip) 安装下载 zip 文件，并按照以下说明进行安装:

1.  移动。zip to SD card 0 > my color screen > Themer > Exported > zip(您的初始位置可能会有所不同。)
2.  打开主题并浏览主题。
3.  在“我的主题”下，选择“导出”
4.  选择“Numero-themer.zip”

如果你不是这个主题的粉丝，设计者也在下面的链接中创建了说明。这是你需要的:

*   [新星发射器](https://play.google.com/store/apps/details?id=com.teslacoilsw.launcher)T3】
*   [Zooper Widget Pro](https://play.google.com/store/apps/details?id=org.zooper.zwpro)
*   [源文件](http://www.android-tuning.sk/wp-content/uploads/Numero-Zooper.zip)
