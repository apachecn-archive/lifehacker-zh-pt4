# 请教专家:关于内向

> 原文：<https://lifehacker.com/ask-an-expert-all-about-introversion-1634824190>

内向和外向的话题一直是讨论的温床——尤其是在网上，我们都在化身背后展示自己，并控制我们如何与他人互动。这些术语到底是什么意思？人存在于两者之间的光谱上吗，你能改变吗？我们去问问心理医生吧。



这里讨论内向的是卡尔顿大学心理学副教授约翰·泽伦斯基。约翰最近咨询了 [Happify](http://my.happify.com/hd/what-every-introvert-needs-to-know-to-be-happy-and-successful/) 来考察快乐和内向之间的关系，并在这里回答你的问题。

想知道这个概念的科学基础，或者只是想找到你在光谱中的位置？约翰下一个小时会在这里，所以问吧！
