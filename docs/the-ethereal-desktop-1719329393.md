# 飘渺的桌面

> 原文：<https://lifehacker.com/the-ethereal-desktop-1719329393>

里克·AAP 将这个充满渴望、神秘的桌面放在一起，融合了一点现代艺术和一些简单的皮肤，让它变得有趣而独特。如果你喜欢它的外观，下面是如何在你的桌面上得到同样的效果。



Ric 的桌面运行的是 Windows，尽管这可能并不明显。以下是设置一切所需的组件:

*   [壁纸](http://depthcore.com/lab/cult/5125/)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   将 [雨量计的优雅皮肤](http://www.deviantart.com/art/Rainmeter-Elegance-1-0-214509019) 时间叠加在壁纸上
*   雨量计的 [VisBubble 皮肤](http://madhoe.deviantart.com/art/VisBubble-for-Rainmeter-488601501) 用于屏幕中央的活动可视化器，它响应音乐
*   屏幕下方的app 启动器
*   [令牌图标包](http://brsev.deviantart.com/art/Token-128429570) 替换了 dock 中的默认图标

这就是这个的全部内容。通常情况下，我们不太喜欢有人物头像的壁纸，但是这一张看起来非常抽象和现代。如果你喜欢它，或者对它有疑问，一定要去 Ric 的 Flickr 页面，链接如下，来发表意见！
