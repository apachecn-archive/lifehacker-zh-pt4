# 城市景观工作区

> 原文:[https://lifehacker.com/the-cityscape-workspace-1722059933](https://lifehacker.com/the-cityscape-workspace-1722059933)

如果你喜欢你的城市(或者你去过的城市)，鸟瞰图可能是你工作站的最佳背景。今天的特色工作区展示了令人惊叹的城市日夜景观。

Watch

帝国大厦和自由塔的图案以及双显示器桌面壁纸都让这个属于 Flickr 用户 Jayson S 的工作空间感觉比房间的实际大小更大、更广阔。书桌的摆放简单而对称，因此这些城市照片非常显眼。

如果你有自己的工作空间可以炫耀，请将它添加到我们的 [Lifehacker 工作空间展示和讲述](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) Flickr 池中，与我们分享。确保您包含的任何照片至少为 640x360。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区。
