# 简约的威尼斯桌面

> 原文：<https://lifehacker.com/the-minimal-venetian-desktop-1629445366>

读者 [乔迪](http://anjody86.kinja.com/) 随着工作日的进行，桌面变得有些凌乱，但在一天开始时，它总是干净的。这是乔迪的桌面在一天中的视图——干净、开放、鼓舞人心，但它隐藏着严肃的力量。这是所有设置的方式。



要像 Jody 一样将您的桌面提交到我们的桌面展示，只需将它发布到您的 Kinja 博客上，标签为“桌面展示”，我们就会看到它！至于 Jody 的桌面，你可能认为它只是一个开放的桌面，桌面上没有图标，但 Jody 的一切都是干净开放的，因为有一个秘密武器在后台运行:Launchy。Jody 使用 Launchy 打开几乎所有东西，Windows 8.1 任务栏中只有四个快捷方式:Explore、Musicbee、Word 和 Chrome。以下是您将此设置变成自己的设置所需的内容:

*   [壁纸](http://www.hdwallpapersphotos.com/venice-river-city-photos/18589/)
*   [令牌灯图标包](http://brsev.deviantart.com/art/Token-128429570) 为任务栏图标
*   [Launchy](http://www.launchy.net/) ，开源的击键启动器(还有 [我们最喜欢的 Windows 应用启动器](http://lifehacker.com/the-best-application-launcher-for-windows-5835770) ，更不用说 [它对其他各种有用的东西](http://lifehacker.com/why-you-should-be-using-an-app-launcher-and-how-to-mak-5963597) )
*   [黑色玻璃平板皮肤](http://alex77456.deviantart.com/art/Black-Glass-Flat-1-2-408455192) 用于发射

这就是你所需要的。没有雨量计，没有定制的皮肤或小部件，没有特殊的调整-只有一个超级干净、开放的桌面，有足够的工作空间，已经过定制，因此打开应用程序和获取您需要的文件和文档超级简单，只需轻轻一击。Jody 在下面的链接中对整个设置进行了更详细的介绍，所以如果你想知道它在日常使用中是如何工作的，请仔细阅读。
