# 尼泊尔桌面

> 原文：<https://lifehacker.com/the-nepali-desktop-1657719128>

读者 [Brisngr](http://roshangautam.kinja.com/) 热爱尼泊尔的自然美景，并将其与 OS X Yosemite 和几个 bersicht 小工具相结合，制作了一个好看、实用的桌面，既有启发性又能提供信息。以下是您也可以设置这一切的方法。

Watch

Brisngr 将这个桌面提交到我们的 [桌面展示区](http://kinja.com/desktop-showcase) ，如果你想在这里看到它的特色，这是你应该做的！这是他用来设置这一切的:

*   [壁纸](https://vrty.org/image/yrx2j0znod/download/wallpaper-1983119.jpg) (最初由安东尼·伯格德 [在 Flickr 上](https://m.flickr.com/#/photos/anthonyburgard/8008126993/) )
*   Yosemite 的“黑暗模式”，将菜单栏和 dock 调暗(在“系统偏好设置”>“通用”中提供)
*   [über sicht](http://tracesof.net/uebersicht/)，之前我们已经介绍过的 OS X 定制套件。
*   用于 bersicht 的 [简单时钟](https://github.com/soberstadt/simple-clock-widget) 小部件
*   bersicht 的 [天气](https://github.com/felixhageloh/weather-widget) 小部件

这就是这个故事的全部内容。我们真的很高兴在展示区看到更多的定制 Mac 台式机。如果你一直在摆弄它，一定要把你的也贴出来！如果你对此有疑问，或者想知道如何让它工作，请访问 Brisngr 的 Kinja 博客(链接如下),提出你的问题并分享你的赞美！

你有一台自己的好看又实用的台式机可以炫耀吗？分享给我们吧！使用标签 [**将它发布到你的个人 Kinja 博客**](http://kinja.com/desktop-showcase) 或添加到我们的 [**Lifehacker 桌面展示并告诉 Flickr pool**](https://www.flickr.com/groups/lifehacker-desktop-showandtell/pool/) 。截图必须至少为 640x360，并请包括您使用的信息，链接到您的壁纸，皮肤和主题，以及任何其他相关的细节。如果你的超棒的桌面吸引了我们的眼球，你可能会成为我们的特色！
