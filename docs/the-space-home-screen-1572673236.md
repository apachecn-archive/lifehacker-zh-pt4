# 太空主屏幕

> 原文：<https://lifehacker.com/the-space-home-screen-1572673236>

Android:如果你一直痴迷于新的宇宙系列，或者你只是第一百万次玩传送门 2[，一个太空主题的主屏幕不会出错。我的彩屏用户T5】佩德罗盖利](https://www.youtube.com/watch?v=4Th_3IM1XJw)T7】已经把你覆盖了。

Watch

这个空间主屏幕是为主题定制的。它不是内置在应用程序本身，但你可以从下面的源代码链接下载 zip 文件，并按照以下说明安装它:

1.  单击下面来源处的绿色“下载主题文件”按钮。
2.  保存。压缩文件到你的手机。
3.  移动。zip to sdcard 0 > my color screen > Themer > Exported > zip*(您的初始位置可能会有所不同。)*
4.  打开主题并浏览主题。
5.  在“我的主题”下，选择“导出”
6.  选择“空间”
