# 倾斜的主屏幕

> 原文：<https://lifehacker.com/the-slanted-home-screen-1691165335>

今天的设计是专为主题打造的，是一个单页主屏幕，将你最重要的信息放在一个看起来很光滑的页面上。

Watch

要在您的设备上获得这种设计，请下载。zip，然后按照以下说明操作:

1.  移动。zip to SD card 0 > my color screen > Themer > Exported > zip(您的初始位置可能会有所不同。)
2.  打开主题并浏览主题。
3.  在“我的主题”下，选择“导出”
4.  选择“倾斜的. zip”
