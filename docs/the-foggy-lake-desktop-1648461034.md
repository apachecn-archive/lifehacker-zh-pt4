# 雾湖桌面

> 原文：<https://lifehacker.com/the-foggy-lake-desktop-1648461034>

有时候定制的小部件是可行的，有时候最小化是最好的方法——我们肯定已经看到了向最小化方向发展的趋势。Flickr 用户[imminant Fate](https://www.flickr.com/photos/116237792@N02/)不久前提交了这个好看的桌面，但它仍然吸引了我们的眼球。以下是你也可以设置它的方法。



这是一个 Windows 桌面，不像我们最近介绍的一些 Linux 桌面，但它看起来仍然干净简单——没有太多分散注意力的屏幕，有足够的工作空间。以下是使您的计算机看起来像这样所需的内容:

*   [壁纸](http://pichost.me/1514257/)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   屏幕上方的 [编码皮肤](http://lilshizzy.deviantart.com/art/Rainmeter-Encoded-v1-2-3-215992941) 为雨量计的时间
*   屏幕下方的 [为天气简单句三皮](http://white-baron.deviantart.com/art/Simplesentencethree-175994306)
*   [英格玛套件](http://enigma.kaelri.com/downloads/) 用于雨量计替换系统图标
*   [Ecqlipse 图标包](http://chrfb.deviantart.com/art/ecqlipse-2-quot-ico-quot-66339090) 替换任务栏图标

公平地说，ImminentFate 已经做了很好的定制和调整，使这个设置看起来如此，所以你的可能看起来不太一样，只是通过安装所有正确的包和调整所有正确的设置。如果你有困难，或者不确定如何让它工作，去他的 Flickr 页面(链接如下)问问他是怎么做到的！
