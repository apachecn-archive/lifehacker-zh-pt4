# 远处的金字塔桌面

> 原文：<https://lifehacker.com/the-distant-pyramid-desktop-1654404411>

我们的朋友 [Dobbie03](http://dobbie03.kinja.com/) 制作了一些非常好看的 Linux 桌面，这款也不例外。在与 Arch 幽会后，他最近搬回了 Manjaro，用一些易于下载的组件构建了这个桌面。以下是您也可以设置它的方法。



Dobbie03 将他的桌面提交到我们的 [桌面展示区](http://kinja.com/desktop-showcase) ，如果你想在这里看到你的桌面，这是你应该做的！这是他用来设置这一切的:

*   来自 Wallhaven 的 [壁纸](http://alpha.wallhaven.cc/wallpaper/53356)([壁纸](http://wallpaperswide.com/pyramids-wallpapers.html) 的替代链接)
*   [Conky](http://conky.sourceforge.net/) ，一款针对 Linux 系统的轻量级系统监视器
*   一个 [自定义的左边栏的 Conky 配置](http://pastebin.com/0XCiPefJ)(dobbie 03 已经很大方的和我们分享了！)
*   [优雅色彩](http://satya164.deviantart.com/art/Gnome-Shell-Elegance-Colors-305966388) 侏儒外壳主题
*   [Zukitre](http://lassekongo83.deviantart.com/art/Zukitre-490751685) GTK 主题
*   屏幕下方的 [板块](https://launchpad.net/plank) 应用启动器，主题设置为透明
*   启动器的 [Numix 圆圈](http://numixproject.org/) 图标

Dobbie03 还指出，他使用和 [奏鸣曲](http://en.linuxreviews.org/Sonata) 来满足他所有的音乐聆听需求，果然 [这首关于两人在音乐会上](http://www.webupd8.org/2009/10/mpd-sonata-powerful-audio-player-for.html) 的曲子概述了他们在一起可以有多么强大。大概就是这样——整个东西看起来很干净，有足够的工作空间，旁边放着合适的工具，以备不时之需。

如果你对这个设置有任何疑问，或者想知道他是如何做到恰到好处的，点击下面的链接去他的 Kinja 博客并得到你的答案。在下面的链接中还有很多其他的设置截图。
