# 生力桌面

> 原文：<https://lifehacker.com/the-san-miguel-desktop-1688949256>

读者 [亲亲刘海！砰！](http://kisskissbangbang.kinja.com/) 想做一个展示他家乡辉煌的桌面，我们认为他已经做到了——太棒了。以下是他如何设置的，以及如何将一块生力放在你的桌面上。



亲亲 Bang！砰！我注意到圣米格尔是联合国教科文组织的世界遗产，所以如果这样看还不够，你应该考虑亲自去看看。如果你现在只有时间享受桌面假期，那么你需要做的就是准备:

*   这张圣米格尔·德·阿连德 的 [照片作为你的壁纸(感谢你提供给我们！)](http://i.imgur.com/LOdo0LA.jpg)
*   用于 OS X 的 [GeekTool](http://projects.tynsoe.org/en/geektool/) 系统监视器和配置实用程序，主要用于右下角的时钟 geeklet
*   OS X 的 [Bowtie](http://bowtieapp.com/) 实用程序，用于在左下角添加正在播放的显示(应用了已传递的主题)
*   用于 OS X 重新排列菜单栏图标的实用程序
*   [cDock](http://www.macupdate.com/app/mac/51149/cdock) 实用程序隐藏 Dock 背景
*   改变应用图标的 [LiteIcon](http://www.freemacsoft.net/liteicon/) 实用程序，以及由 [Andrew](http://twitter.com/andrevvm) 和 [Kara](http://kara-z.com/) 提供的 [.app.icns](http://appicns.com/) 图标包

显然，这个特殊的设置是在 OS X，但你可以在 Windows 或 Linux 中通过一些我们之前在以前的特色桌面中展示过的调整来实现这一点，比如在屏幕上使用 Rainmeter 或在 Linux 中使用 Conky 和 Docky。不管怎样，如果你是在你自己的 Mac 电脑上解决这个问题，那就去亲亲吧！砰！分享你有多喜欢成品——它看起来真的很棒。
